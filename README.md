# English Paraphrase
「 paraphrase 」とは、難しい言葉を簡潔な表現に書き換えるとという意味です。
English Paraphraseは、ユーザーが英文や英単語を投稿し、それらと類似した意味で様々な表現する、そして共有するアプリです。
